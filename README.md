# Dotfiles

Rebirth of https://github.com/riseshia/legacy-dotfiles

## Installation

```
$ curl -sfSL https://raw.githubusercontent.com/riseshia/dotfiles/master/install.sh | bash
```

## Claude Code Plugins

```
/plugin marketplace add riseshia/dotfiles
```

## Usage

```
git pull        # Update source (symlinks reflect changes immediately)
./setup.sh      # Re-run after adding new packages or symlinks
```
